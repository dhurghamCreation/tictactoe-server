# React Tic-Tac-Toe (TypeScript + Vite)

A minimal Tic-Tac-Toe game built with React + TypeScript and Vite.

## Quick start

1. Install dependencies:
```bash
npm install
# or
yarn
```

2. Run dev server:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

This repo includes TypeScript sources and one small JavaScript utility to demonstrate mixed JS/TS usage.
